# Slide screenshots

Drop your Canva PNG exports here with these exact filenames:

- `home.png`     → first slide (Slides / hivetalk.org)
- `talks.png`    → second slide (talks list)
- `projects.png` → third slide (Vanilla / Honey / Lookup / Relays)

Recommended export:

- **Aspect**: 1366×768 (matches the original Canva canvas) OR any 16:9 size
- **Format**: PNG (or JPG for smaller files)
- **Resolution**: 2x (2732×1536) for crisp retina display

The page will auto-scale them to fit the viewport. Link hotspots are positioned
in percentages in `src/pages/index.astro`, so they stay aligned at any size — as
long as the screenshots are 16:9 and cropped exactly like the originals.
